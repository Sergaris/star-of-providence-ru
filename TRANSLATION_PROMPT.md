# Star of Providence — Перевод на русский

## Задача

Перевести CSV-файл локализации из каталога `localization/` этого репозитория. Каждый файл переводится **независимо**, по одному за раз.

**Непереведённая строка** — та, где значение в колонке `ZHS` совпадает с колонкой `EN` (английский фоллбэк). Твоя задача: заменить английский текст в `ZHS` на русский перевод.

---

## Контекст игры

**Star of Providence** (ранее Monolith) — top-down roguelite-шутер на GameMaker. Мрачная атмосфера, подземное сооружение, антропоморфные NPC с неформальной речью. Игрок исследует этажи, собирает оружие с ключевыми словами, сражается с боссами.

Жанр: rogue-lite, dungeon crawler, bullet hell.

---

## Формат CSV

- **Кодировка**: UTF-8 with BOM (`utf-8-sig`)
- **Заголовок**: `ID,Comments,EN,DE,FR,ES,PTBR,ZHS,JA`
- **Исходная колонка**: `EN` (индекс 2) — читать отсюда
- **Целевая колонка**: `ZHS` (индекс 7) — писать перевод сюда
- **Остальные колонки**: НЕ ТРОГАТЬ

### Типы строк

| Тип | Признак | Действие |
|-----|---------|----------|
| Данные | Числовой ID в первой колонке | Перевести ZHS |
| Комментарий | Пустой ID, текст в Comments | Оставить как есть |
| Пустая | Все поля пустые | Оставить как есть |
| Заголовок | Первая строка файла | Оставить как есть |

### Как определить непереведённую строку

```
ZHS == EN  →  не переведено (нужен перевод)
ZHS != EN  →  уже переведено (не трогать)
```

## Спецсимволы — СОХРАНЯТЬ В ТОЧНОСТИ

| Символ | Значение | Пример |
|--------|----------|--------|
| `#` | Перенос строки | `двойной урон#на 3 секунды` |
| `##` | Двойной перенос / абзац | `описание##детали` |
| `/c1` `/c0` | Цветной текст (начало/конец) | `/c1ключевые слова/c0` |
| `/c2` `/c4` `/c5` | Другие цвета | `/c2всегда/c0` |
| `/f1` `/f0` | Форматирование (курсив) | `/f1*вздох*/f0` |
| `/p1` `/p2` | Пауза в тексте | `/p1./p1./p1.` |
| `%var%` | Переменная | `%nr%`, `%loop%`, `%tier%`, `%deathsource%` |
| `A%` `D%` `P%` | Переменные процентов | `+A% макс. боезапас` |
| `_` | Завершающий символ (машинный текст) | `очистка в процессе…_` |

**Правило**: количество тегов `/c`, `/f`, `/p` и переменных `%..%` в переводе ДОЛЖНО совпадать с оригиналом. Если в EN есть `/c1.../c0`, в ZHS тоже должно быть `/c1.../c0`.

## Стиль перевода

### Обязательно

1. **Краткость** — не длиннее оригинала. Пиксельная игра, маленький шрифт. Если русский вариант длинный — ищи короткий синоним.
2. **Нижний регистр** — игра использует lowercase. Перевод тоже, кроме аббревиатур (ОЗ, ЭЛТ).
3. **Неформальный тон** — лаконично, местами иронично. Без канцелярита и лишних слов.
4. **Единообразие** — используй глоссарий ниже. Если термин уже переведён в другом файле — переводи так же.

### Запрещено

- Менять колонки кроме ZHS
- Менять строки-комментарии (пустой ID)
- Менять пустые строки-разделители
- Удалять или добавлять строки
- Менять заголовок файла
- Переводить уже переведённые строки (ZHS ≠ EN)

## Примеры перевода по типам файлов

### Названия (name) — коротко, ёмко

```
unstable catalyst → нестабильный катализатор
ancient runestone → древний рунный камень
petri dish → чашка Петри
```

### Описания (description / tooltip) — краткие фразы с `#`

```
gain extra rewards#when replacing weapons → доп. награды#при замене оружия
immunity to lightning, frost,#and hazards → иммунитет к молнии, морозу#и опасностям
```

### Диалоги (gossip) — разговорная речь NPC

```
found anything interesting down there? → нашёл что-нибудь интересное там внизу?
don't wanna get my fur wet anyway. → не хочу мочить шерсть, в любом случае.
nifty, eh? → удобно, а?
```

### Смерти (death) — ирония, сочувствие

```
you got squashed? → тебя раздавило?
how'd you even? → как вообще?
what a terrible way to go. → жуткая смерть.
```

### Титры (credits) — ТОЛЬКО заголовки, имена НЕ переводить

```
programming: → программирование:
art and direction: → арт и руководство:
special thanks → особая благодарность
```

### Клавиши (keyboard_keys) — НЕ ПЕРЕВОДИТЬ

Файлы `keyboard_keys.csv` и `keyboard_keys_switch.csv` исключены из локализации.

## Процедура для каждого файла

1. **Прочитай** файл из `localization/<имя>.csv`
2. **Найди** непереведённые строки: у них `ZHS == EN`
3. **Переведи** каждую строку: замени значение ZHS на русский перевод
4. **Проверь**:
   - Все спецсимволы (`/c`, `/f`, `/p`, `#`, `%var%`, `_`) сохранены
   - Перевод не длиннее оригинала более чем на 20%
   - Тон и стиль соответствуют игре
5. **Не трогай**: заголовок, комментарии, пустые строки, другие колонки, уже переведённые строки

### Пример замены

Исходная строка в файле:
```
5,,immunity to lightning,immunität gegen blitz,immunité à la foudre,inmunidad a electricidad,imunidade a relâmpagos,immunity to lightning,雷への耐性
```

После перевода:
```
5,,immunity to lightning,immunität gegen blitz,immunité à la foudre,inmunidad a electricidad,imunidade a relâmpagos,иммунитет к молнии,雷への耐性
```

Изменилось только значение в позиции ZHS (8-я колонка, индекс 7 с нуля).
